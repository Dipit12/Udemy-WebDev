## Intro to EJS
    - EJS stands for Embedded JS

## Different Types of EJS tags
    - <%= variable %> -> JS output
    - <%  console.log("") %> -> JS Execute, but doesnot give any output
    - <%- <h1>Hello </h1>  %> -> Render HTML
    - <%%  %%>
    - <%#   %>   -> Stop execution: Basically comments
    - <%- include("<FILENAME>") %>   -> Insert another EJS file
